# BMO

## Role

Software Engineering Intern on BMO's NARP Platforms Tech team, January to
April 2026.

## File Monitor System

I led and developed a file monitoring system and dashboard that tracks file
presence, metadata, and SLA compliance, and sends notifications when an SLA is
at risk. It's built with DQL queries in Dynatrace plus Python and JSON in
Ansible. It cut manual escalation time by about 60% and reduced SLA breaches,
which prevents fines.

## TSYS Monitoring Dashboard

I helped develop a TSYS monitoring dashboard that gives end-to-end visibility
into TSYS data flows, AWS API and MQ volumes, and consumer usage patterns,
using AWS Lambda CloudWatch logs for the data analysis.
